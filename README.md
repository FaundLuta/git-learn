# 🎓 Git Learn

🔗 **Live Demo:** [git-learn-lemon.vercel.app](https://git-learn-lemon.vercel.app/)

An interactive, responsive single-page web application designed to help developers master Git and GitHub commands natively from their workspace.

## 🚀 Features
* **14 Guided Lessons**: Spans across Setup, Basic Commits, History management, Branching, Undoing mistakes, and Remote operations.
* **Interactive Step Tracking**: Checklists to mark your progress as you complete hands-on terminal steps.
* **One-Click Code Copy**: Instantly copy terminal commands to execute them locally.
* **Visual Progress Tracking**: Real-time progress bar tracking lesson completion.

## 🛠️ Tech Stack
* **Frontend**: Vanilla HTML5, CSS3 (Google Fonts integration), and modern asynchronous JavaScript.
* **Theme**: Clean UI styled with standard Google-inspired design palettes.

## 💻 How to Run Locally
1. Clone this repository:
```bash
   git clone https://github.com/FaundLuta/git-learn.git
```
2. Navigate into the project directory:
```bash
   cd git-learn
```
3. Open `index.html` directly in your browser, or serve it locally:
```bash
   npx serve .
```
